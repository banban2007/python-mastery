# Python Learning Journey 🚀

ဒီ Repository ကတော့ ကျွန်တော် Python Programming ကို အခြေခံကနေ Django Framework အထိ စနစ်တကျ လေ့လာနေတဲ့ မှတ်တမ်းဖြစ်ပါတယ်။ Folder တစ်ခုချင်းစီမှာ သင်ခန်းစာအလိုက် Code တွေနဲ့ Example တွေကို လေ့လာနိုင်ပါတယ်။

## 📑 မာတိကာ (Table of Contents)

1.  **Basics**
    * `1_sequence` & `2_sequence` - အခြေခံ Sequence များ
    * `3_looping` - ပတ်လမ်းကြောင်းများ (For & While Loops)
    * `4_string` - စာသားပိုင်းဆိုင်ရာ လုပ်ဆောင်ချက်များ
2.  **Intermediate**
    * `5_data_structure` - List, Tuple, Dictionary, Set
    * `6_functions` - Function ရေးသားနည်းများ
    * `7_modules` & `9_packages` - Module နှင့် Package အသုံးပြုပုံ
    * `8_python_file_and_folder` - File Handling (ဖိုင်အဖွင့်အပိတ်)
3.  **Advanced & Specialized**
    * `10_exception` - Error Handling
    * `11_date_time` - အချိန်နှင့် နေ့စွဲများ
    * `12_OOP` - Object-Oriented Programming
    * `13_database` - Database ချိတ်ဆက်မှုများ
    * `14_django` - Web Development (Django Framework)
4.  **Practice**
    * `python mini project` - လက်တွေ့လုပ်ဆောင်ထားသော Project ငယ်များ
    * `testing` - Code စမ်းသပ်မှုများ

## 🛠 လိုအပ်ချက်များ (Requirements)
* Python 3.x
* VS Code (သို့မဟုတ် ကြိုက်နှစ်သက်ရာ Editor)
* Django (Folder 14 အတွက်သာ)
